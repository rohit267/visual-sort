# Visual Sort

Visual Sort is an sorting algorithm visualizer made using reactjs

## Algorithms

- merge Sort
- selection Sort 
- bubble Sort
- insertion Sort
- quick Sort  "yet to implement"
- shell Sort  "yet to implement"
- counting Sort "yet to implement"
